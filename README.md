# 🚀 Smart Document Automation System

> AI-powered document classification and routing system for colleges

---

## 📌 Overview

This project automates the process of handling documents by:
- Extracting text from uploaded files
- Classifying them using Machine Learning
- Routing them to the correct department

---

## 🎯 Problem Solved

Manual document sorting in colleges is:
- Time-consuming
- Error-prone
- Inefficient

👉 This system automates the entire workflow.

---

## ⚙️ Features

✅ Upload PDF documents  
✅ Automatic text extraction  
✅ Intelligent classification  
✅ Department-based routing  
✅ Clean and simple UI  

---

PDF → Text Extraction → TF-IDF → Naive Bayes → Category → Routing
