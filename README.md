# Smart Document Automation 📄🤖

## Overview

This project is an AI-based system that automatically processes documents by extracting text, classifying their type, and routing them to the appropriate department.

## Problem Statement

In colleges, documents like fee receipts, certificates, and applications are manually sorted and processed. This is time-consuming and prone to errors.

## Solution

This system automates the process using Machine Learning and NLP techniques:

* Extract text from documents (PDFs)
* Classify documents into categories
* Route them to the correct department
* Store metadata for tracking

* (Optional) Streamlit (UI)

## Project Workflow

1. Upload document
2. Extract text
3. Preprocess text
4. Convert to features (TF-IDF)
5. Classify using ML model
6. Route to department
